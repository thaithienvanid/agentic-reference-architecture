# Execution ontology

> **Status: Compatibility page.** The normative model is [RFC — Execution model](/rfc/execution-model).

```text
ExecutionPlanRun
└── WorkflowRun
    └── ActivityRun
        ├── ActivityAttempt
        ├── ExecutionBranch
        ├── Iteration
        └── Effect
            └── Invocation

WorkerLease
ExperimentRun -> ExperimentVariant -> ExperimentTrial -> WorkflowRunRef
```

Key rules:

- `Run` is reserved for independently durable logical executions.
- An `Effect` is one logical external or nondeterministic operation.
- An `Invocation` is one concrete call for that effect.
- Worker resumption creates a new `WorkerLease`, not automatically an activity retry.
- Changed prompt, context, arguments, tool, or objective creates a new effect.
- `DeliberationRound` is a domain label mapped to an `Iteration`.
- Model A versus Model B is a variant comparison, not a round.
