## How to use guides

Use a guide when you are deciding how to implement or operate a requirement. Use [Reference](/reference/index) for exact names and contracts, [Patterns](/patterns/index) for reusable compositions, and [ADRs](/decisions/adrs) for the reasons behind consequential choices.
