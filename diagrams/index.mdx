## 7. Complete run sequence

```mermaid
sequenceDiagram
  participant Client
  participant App
  participant Runtime as Runtime Service
  participant Journal as Platform Run Journal
  participant Model
  participant Policy
  participant Human
  participant Tool

  Client->>App: Start run
  App->>Journal: run.started
  Runtime->>Journal: effect.planned (model)
  Runtime->>Model: Invocation 1
  Model-->>Runtime: Action proposal
  Runtime->>Policy: Evaluate proposed tool Effect
  Policy-->>Runtime: Require approval
  Runtime->>Journal: approval.requested + checkpoint
  Human->>App: Approve action digest
  App->>Runtime: Resume signal
  Runtime->>Policy: Revalidate
  Runtime->>Journal: effect.planned (tool)
  Runtime->>Tool: Invocation with idempotency key
  Tool-->>Runtime: Result
  Runtime->>Journal: effect.completed + run.completed
  Runtime-->>Client: Result
```
