```mermaid
  participant Human

  App->>Journal: RunStarted

  Model-->>Runtime: Action proposal
  Runtime->>Policy: Evaluate
  Policy-->>Runtime: Require approval
  Runtime->>Journal: ApprovalRequested + checkpoint

  App->>Runtime: Resume signal

  Tool-->>Runtime: Result
  Runtime->>Journal: ToolCompleted + RunCompleted
  Runtime-->>Client: Result
```
