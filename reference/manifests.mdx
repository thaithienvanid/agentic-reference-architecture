# Manifest profile

> **Status: Reference profile.** Manifests are declarative control-plane inputs. Production compilation resolves references to exact immutable versions and digests.

Money follows [Canonical conventions](/reference/conventions) and is serialized as decimal strings.

## Workflow version

```yaml
apiVersion: agentic.platform/v1
kind: WorkflowVersion
metadata:
  name: procurement-order
  version: 3.0.0
spec:
  inputSchema: ProcurementTask@2
  outputSchema: ProcurementResult@3
  stateSchema: ProcurementRunState@2
  activities:
    - id: load-context
      type: task
      outputSchema: ProcurementContextRefs@1
    - id: research
      type: agent
      agent: procurement-research-agent@2.0.0
    - id: validate
      type: task
      input:
        research:
          from: activities.research.output
    - id: approve
      type: human
      policy: high-value-purchase-approval@5.0.0
    - id: create-order
      type: tool
      tool: purchase-order.create@4.2.1
  dependencies:
    - [load-context, research]
    - [research, validate]
    - [validate, approve]
    - [approve, create-order]
```

    action: tool.invoke

      totalValueEur: { greaterThanOrEqual: 100000 }

        - runId
