## Event categories

- Domain events carry business meaning.
- Execution events describe run, activity, effect, invocation, and lease lifecycles.
- Integration events cross bounded contexts through the outbox.
- Audit events carry authoritative security and governance evidence.
- Telemetry signals support operations and may be sampled.

## State transaction

```text
BEGIN
  verify expected state version and fencing token
  append canonical events
  update run-state projection
  update effect, invocation, and budget ledgers
  write integration outbox
COMMIT
```

Historical events are immutable. Compatible evolution uses explicit schema versions and readers or upcasters.
